Detailed Course Syllabus

1. Introduction to Programming

Module No. Module Name Topics Covered CO Mapping
1 Introduction to Programming & Basics History of Programming, Programming Paradigms, Introduction to Python/C/Java, Basic Syntax, Variables, Data Types, Operators CO1
2 Control Structures Conditional Statements (if-else, switch-case), Loops (for, while, do-while), Nested Control Structures CO2
3 Functions and Modular Programming Functions, Recursion, Scope & Lifetime of Variables, Function Overloading, Lambda Functions CO3
4 Data Structures and Algorithms Lists, Arrays, Strings, Dictionaries, Stacks, Queues, Sorting & Searching Algorithms CO4
5 File Handling and Advanced Topics File I/O, Exception Handling, Introduction to Object-Oriented Programming Concepts CO5

2. Data Structures

Module No. Module Name Topics Covered CO Mapping
1 Introduction to Data Structures Definition, Classification, Abstract Data Types (ADTs), Importance of Data Structures CO1
2 Linear Data Structures Arrays, Linked Lists (Singly, Doubly, Circular), Stacks, Queues (Normal, Circular, Priority) CO2
3 Non-Linear Data Structures Trees (Binary Trees, BST, AVL, B-Trees), Graphs (Representation, BFS, DFS, Applications) CO3
4 Sorting and Searching Algorithms Sorting (Bubble, Selection, Insertion, Merge, Quick, Heap), Searching (Linear, Binary) CO4
5 Advanced Topics and Applications Hashing, Memory Management, Advanced Graph Algorithms, Real-world Applications CO5

3. Artificial Intelligence

Module No. Module Name Topics Covered CO Mapping
1 Introduction to AI Definition, History, Applications, AI vs. Machine Learning vs. Deep Learning, Intelligent Agents CO1
2 Problem Solving and Search State Space Search, Uninformed Search (BFS, DFS), Informed Search (A*, Greedy), Constraint Satisfaction Problems CO2
3 Knowledge Representation and Reasoning Logic-based Representation, Semantic Networks, Frames, Expert Systems, Probabilistic Reasoning CO3
4 Machine Learning Fundamentals Supervised Learning (Regression, Classification), Unsupervised Learning (Clustering), Reinforcement Learning Basics CO4
5 Advanced AI Topics and Applications Natural Language Processing, Computer Vision, Robotics, Ethics in AI, AI in Industry Applications CO5
